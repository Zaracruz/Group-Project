HTML

HTMl is the standard markup language for Web Pages. HTML stand for Hyper Text Markup Language. It descirbes the strcuture of a webpage, it consists of a series of elements, these elements tell the browser how to display the content.

Elements of HTML:
  • HTML Element
  • Head Element
  • Title Element
  • Base Element
  • Link Element
  • Meta Element

Html is used to create webpages but does experience limitations
when it comes to fully responsive components. Therfore html should only
be used to add text elements and strutcture them within a page. For more
complex features, html can be combined with cascading style sheets(CSS) and javascript(JS).

CSS

CSS is an acroynym for Cascading Style Sheets. Css is a computer lanuage for laying out and styling web pages. It is usually a textfile srtuctured using a markup language, the most commonly used markup is HTML. CSS can control the style of multiple web pages at once, or you can create seperate style sheets for different pages.

Types of CSS:

External Internal Inline
A file that HTML files will link to Specified at the beginning of an HTML document Written for a specific element in the HTML document.

The Big Three

  1. Inheritance
  2. Specificty
  3. Cascade
Inheritance

some CSS property values set on parent elements are inherited by
their child elements, and some aren't. For example, if you set a
color and font-family on an element, every element inside it will
also be styled with that color and font, unless you've applied
different color and font values directly to them.

This very page uses Inheritance css. As you can tell it isn't very flattering sometimes and can be difficult to keep track of.

Specificty

Specificity is the algorithm that the browser uses to decide
which property value is applied to an element. If multiple style blocks
have different selectors that configure the same property with different
values and target the same element, specificity decides the property value
that gets applied to the element. Specificity is basically a measure of how specific
a selector's selection will be:
  • An element selector is less specific; it will select all elements of that type that appear on a page, so it has less weight. Pseudo-element selectors have the same specificity as regular element selectors.
  • A class selector is more specific; it will select only the elements on a page that have a specific class attribute value, so it has more weight. Attribute selectors and pseudo-classes have the same weight as a class.
